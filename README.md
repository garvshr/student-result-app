* Student Result Management System (Mini Project)

A minimal, clean, and elegant Student Result Management System built using React, Tailwind CSS, Axios (AJAX), and JSON Server.
This project implements CRUD operations and demonstrates core web development concepts in a practical, real-world context.

* Objective 

To integrate all foundational web development concepts — HTML, CSS, JavaScript, async operations, REST APIs, JSON handling, and UI design — into a working React application capable of managing student results with full CRUD functionality.
